# Superstore Sales Dashboard

## Overview

This project analyzes the Superstore dataset using Google Sheets. The dashboard provides insights into sales performance, profit distribution, and sales trends over time.

## Features

* Data cleaning and duplicate removal
* Pivot Tables for analysis
* Sales by Category (Bar Chart)
* Profit by Category (Column Chart)
* Sales Trend Over Time (Line Chart)
* Conditional Formatting for Sales and Profit
* Interactive Dashboard design

## Tools Used

* Google Sheets
* Microsoft Excel
* GitHub

## Dataset

Superstore Dataset from Kaggle.

## Insights

* Technology category generated the highest sales and profit.
* Sales trends varied across different months and years.
* Conditional formatting helped identify high and low performers quickly.
